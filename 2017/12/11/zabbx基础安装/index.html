一、安装部署zabbix 3.4

Zabbix 是由 Alexei Vladishev 开发的一种网络监视、管理系统，基于 Server-Client 架构。可用于监视各种网络服务、服务器和网络机器等状态。

1.zabbix基础说明

1.1 zabbix使用说明

  • zabbix3.4软件中文使用手册

  • zabbix3.4版本安装包路径

1.2 zabbix 3.4版本说明

Zabbix Server端安装的软件：

  • zabbix-server-mysql
  • zabbix-get
  • mysql
  • zabbix-web
  • zabbix-web-mysql

Zabbix Clinet 端安装的软件

  • zabbix-agent
  • zabbix-sender

2.zabbix系统环境配置

2.1 系统环境

# cat /etc/redhat-release
CentOS Linux release 7.4.1708 (Core)

2.2 永久修改主机名

# hostnamectl --static set-hostname zabbix-server

2.3 关闭防火墙及selinux

# systemctl stop firewalld.service
# systemctl daemon-reload
# systemctl disable firewalld.service

# sed -i 's/SELINUX=enforcing/SELINUX=disabled/g' /etc/selinux/config
# grep SELINUX=disabled /etc/selinux/config
# setenforce 0
# getenforce
  Disabled

3.数据库的安装与配置

3.1 MariaDB概述

MariaDB数据库管理系统是MySQL的一个分支，主要由开源社区在维护，采用GPL授权许可。
开发这个分支的原因是：甲骨文公司收购了MySQL后，有将MySQL闭源的潜在风险，因此社区采用分支的方式来避开这个风险。MariaDB的目的是完全兼容MySQL，包括API和命令行，使之能轻松成为MySQL的代替品。

3.2 安装mariadb

# yum install mariadb-server mariadb -y
# rpm -qa |grep maria
mariadb-5.5.56-2.el7.x86_64
mariadb-server-5.5.56-2.el7.x86_64
mariadb-libs-5.5.56-2.el7.x86_64

3.3 配置数据库服务

# vim /etc/my.cnf
添加如下两行：
innodb_file_per_table=on
skip_name_resolve=on

3.4 启动mariadb数据库并设置开机自启动

# systemctl start mariadb
# systemctl enable mariadb

3.5 设置mariadb数据库的安全配置

# mysql_secure_installation  //根据需求设置密码
password:test
# mysql -uroot -ptest //登录mysql

3.6 创建数据库zabbix数据库并授权用户

创建zabbix数据库，并创建zbuser用户具有库的访问权限。

> CREATE DATABASE  zabbix  DEFAULT CHARACTER SET utf8 COLLATE utf8_bin;
> GRANT ALL PRIVILEGES ON zabbix.* TO zbuser@localhost IDENTIFIED BY 'test';
> FLUSH PRIVILEGES ;
# mysql -uzbuser -ptest zabbix //登录测试

4.安装配置zabbix.3.4

4.1 zabbix相关软件包的安装

4.1.1 安装zabbix相关的yum源

# rpm -ivh http://repo.zabbix.com/zabbix/3.4/rhel/7/x86_64/zabbix-release-3.4-1.el7.centos.noarch.rpm

4.1.2 安装zabbix相关包

zabbix服务端需要安装：zabbix-server-mysql zabbix-web-mysql zabbix-get
zabbix被监控主机端安装的软件：：zabbix-agent, zabbix-sender

# yum install zabbix-server-mysql zabbix-web-mysql zabbix-get zabbix-agent zabbix-sender -y
# rpm -qa |grep zabbix
 zabbix-web-mysql-3.4.4-2.el7.noarch
 zabbix-get-3.4.4-2.el7.x86_64
 zabbix-web-3.4.4-2.el7.noarch
 zabbix-agent-3.4.4-2.el7.x86_64
 zabbix-release-3.4-1.el7.centos.noarch
 zabbix-server-mysql-3.4.4-2.el7.x86_64
 zabbix-sender-3.4.4-2.el7.x86_64

4.2 zabbix相关数据导入数据库

# cd /usr/share/doc/zabbix-server-mysql-3.4.4/
# zcat create.sql.gz |mysql -uzbuser -ptest zabbix

4.3 修改数据库相关配置

# vim /etc/zabbix/zabbix_server.conf
# grep -n '^'[a-Z] /etc/zabbix/zabbix_server.conf
38:LogFile=/var/log/zabbix/zabbix_server.log
49:LogFileSize=0
72:PidFile=/var/run/zabbix/zabbix_server.pid
82:SocketDir=/var/run/zabbix
91:DBHost=localhost
101:DBName=zabbix
117:DBUser=zbuser
125:DBPassword=test
132:DBSocket=/var/lib/mysql/mysql.sock
316:SNMPTrapperFile=/var/log/snmptrap/snmptrap.log
434:Timeout=4
476:AlertScriptsPath=/usr/lib/zabbix/alertscripts
486:ExternalScripts=/usr/lib/zabbix/externalscripts
522:LogSlowQueries=3000

4.4 启动zabbix server并设置开机启动

# systemctl enable zabbix-server
# systemctl start zabbix-server

4.5 编辑Zabbix前端PHP配置,更改时区

# vim /etc/httpd/conf.d/zabbix.conf
php_value date.timezone Asia/Shanghai
php_value date.timezone Asia/Shanghai

4.6 启动httpd并设置开机启动

# systemctl start httpd
# systemctl enable httpd

4.7 图形界面安装配置zabbix

浏览器输入http://server_ip/zabbix,根据图形引导信息,配置zabbix信息。

zabbix初始登录账户和密码为： Admin zabbix

5.zabbxi-agent安装及配置

5.1 安装zabbxi-agent

# yum install zabbix-agent  zabbix-sender -y
# grep -n '^'[a-Z] /etc/zabbix/zabbix_agentd.conf

 13:PidFile=/var/run/zabbix/zabbix_agentd.pid
 32:LogFile=/var/log/zabbix/zabbix_agentd.log
 43:LogFileSize=0
 95:Server=127.0.0.1
 136:ServerActive=127.0.0.1
 147:Hostname=zabbix-server
 265:Include=/etc/zabbix/zabbix_agentd.d/*.conf

Server：被动模式，允许哪台服务器连接Agent。
ServerActive：主动模式，向哪台服务器传送数据。

5.2 启动zabbix-agent

# systemctl start zabbix-agent
# systemctl enable zabbix-agent
