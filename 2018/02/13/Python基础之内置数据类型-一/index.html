Python基础之内置数据类型(一)

发表于 2018-02-13 | 分类于 python基础知识 | | 本文总阅读量 次
字数统计: 1,207 | 阅读时长 ≈ 5

前言

在 Python 中， 每个值都有一种数据类型，但您并不需要声明变量的数据类型。那该方式是如何运作的呢？Python 根据每个变量的初始赋值情况分析其类型，并在内部对其进行跟踪。

内置数据类型分类

  • Booleans［布尔型］ 或为 True［真］ 或为 False［假］
  • Numbers［数值型］ 可以是 Integers［整数］（1 和 2）、Floats［浮点数］（1.1 和 1.2）、Fractions［分数］（1/2 和 2/3）；甚至是 Complex Number［复数］
  • Strings［字符串型］ 是 Unicode 字符序列，例如： 一份 html 文档
  • Bytes［字节］ 和 Byte Arrays［字节数组］， 例如: 一份 jpeg 图像文件
  • Lists［列表］ 是值的有序序列
  • Tuples［元组］ 是有序而不可变的值序列
  • Sets［集合］ 是装满无序值的包裹
  • Dictionaries［字典］ 是键值对的无序包裹

布尔类型

布尔类型或为真或为假。Python 有两个被巧妙地命名为 True 和 False 的常量，可用于对布尔类型的直接赋值。表达式也可以计算为布尔类型的值。数字 0、None、以及元素为空的容器类对象都可视作 bool False，反之为 True。

>>> bool(0)
False
>>> bool(None)
False
>>> bool([])
False
>>> bool({})
False
>>> bool(1)
True
>>> bool([1,2])
True

数字类型

python本身支持整数以及浮点数。你可以对这些数字进行下表中的计算。

整数

任何仅含数字的序列在 Python 中都被认为是整数

>>> 5
5

Python还支持运算次序，因此你可在同一个表达式中使用多种运算。你还可以使用括号来修 改运算次序，让Python按你指定的次序执行运算，如下所示:

>>> 2 + 3 * 4
14
>>> (2 + 3) * 4
20

在这些示例中，空格不影响Python计算表达式的方式，它们的存在旨在让你阅读代码时，能迅速确定先执行哪些运算。

浮点数

Python将带小数点的数字都称为浮点数。大多数编程语言都使用了这个术语，它指出了这样一个事实:小数点可出现在数字的任何位置。

>>> 0.1 + 0.1
0.2
>>> 2 * 0.1
0.2

但需要注意的是，结果包含的小数位数可能是不确定的:

>>> 3 * 0.1
0.30000000000000004
>>> 0.2 + 0.1
0.30000000000000004
>>> 4 * 0.1
0.4

字符串

字符串是由多个字符组成的序列。在Python中，用引号括起的都是字符串，字符串定义简单自由，可以是单引号、双引号或者三引号。字符串是不可变序列（immutable, sequence）类型，默认存储 Unicode 文本。

字符串使用

>>> s = "chenfanlinux博客"
>>> len(s)
14

>>> print(ascii(s))
'chenfanlinux\u535a\u5ba2'

内置函数 ascii 将目标转换为可打印 ASCII 字符组成的字符串。

构建字符串很容易，单引号、双引号，以及跨行的三个引号。

>>> "ab'c"  #双引号
"ab'c"
>>> 'ab"c"' #单引号
'ab"c"
>>> """ abc #多行
efd
hij
"""
' abc\nefd\nhij\n'
>>> 'ab\'c' #引号转义
"ab'c"

可在字面量前添加特殊指示符。

>>> print(r"abc\nd")  # raw string，禁用转义。
abc\nd
>>> type(b"ab")
<class 'bytes'>
>>> type(u"ab")
<class 'str'>

str() 类型转换

>>> "abc" + str(2.2)
'abc2.2'

合并字符串

format格式化输出(推荐使用)

>>> "谁的博客很酷: {}".format("陈凡")
'谁的博客很酷: 陈凡'

>>> s ="""你的名字是: {};
你的工资是多少: {};
你是做什么工作的: {}
""".format("陈凡","不告诉你","运维")
>>> print(s)
你的名字是: 陈凡;
你的工资是多少: 不告诉你;
你是做什么工作的: 运维

>>> "你喜欢什么音乐:{0} 你的偶像是谁:{1}".format("戒指","你猜")
'你喜欢什么音乐:戒指 你的偶像是谁:你猜'

% 字符串格式化

>>> msg = "my name is %s and weight is %d kg!" %("chenfan",120)
>>> print(msg)
my name is chenfan and weight is 120 kg!

+

>>> "欢迎来到:" + "www.chenfanlinux.org"
'欢迎来到:www.chenfanlinux.org'

split() 分割

>>> (man,said) = "chenfan : welcome to my bolg:www.chenfanlinu.org".split(':',1)
>>> print(man)
chenfan
>>> print(said)
 welcome to my bolg:www.chenfanlinu.org

strip() 移除头尾的字符串

>>> " www.chenfanlinux.org ".strip()  # 注意只能移除头部和尾部字符串
'www.chenfanlinux.org'

>> "www.chenfanlinux. org".strip() # 这种空格无法移除
www.chenfanlinux. org'

>>> "www.chenfanlinux.org".strip("www.")
'chenfanlinux.org'

join() 合并列表

>>> path = ["/etc", "nginx", "nginx.conf" ]
>>> '/'.join(path)
'/etc/nginx/nginx.conf'

find 查找子串

查找到返回该子串在原字符串中的索引位置，如果无法找到，find方法会返回值-1

>>> s = "www.chenfanlinux.org"
>>> s.find("chenfan")  #访问索引位置
4
>>> s.find("chenfanfan") # 未找到返回-1
-1

字符串切片

存在索引的数据类型，包括字符串,列表和元组都可以做切片操作

>>> s = "www.chenfanlinux.org"
>>> s[0]
'w'
>>> s[0:5]
'www.c'
>>> s[-1]
'g'
