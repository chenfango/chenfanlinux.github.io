python版本管理工具-pyenv

发表于 2018-04-11 | 分类于 python基础知识 | | 本文总阅读量 次
字数统计: 334 | 阅读时长 ≈ 2

Python版本管理工具pyenv，pyenv让你轻松切换多个版本的Python。它很简单，不引人注意，并且遵循UNIX的单一目的工具的传统，可以很好地完成一件事。

相关地址：

  • pyenv github地址: http://github.com/yyuu/pyenv
  • pyenv 安装器: https://github.com/pyenv/pyenv-installer

安装pyenv

安装git

$ yum -y install  git

安装pyenv

$ curl -L https://github.com/pyenv/pyenv-installer/raw/master/bin/pyenv-installer | bash

配置环境变量

$ vim /etc/profile.d/pyenv.sh
export PATH="/root/.pyenv/bin:$PATH"
eval "$(pyenv init -)"
eval "$(pyenv virtualenv-init -)"
$ source /etc/profile

安装python

安装编译工具

$ yum install -y   gcc make patch

安装依赖

$ yum install -y  gdbm-devel openssl-devel sqlite-devel zlib-devel bzip2-devel readline-devel

安装python 3.5.2

$ pyenv install 3.5.2 # 这种方式会下载官方源，可能会比较慢
$ mkdir   ~/.pyenv/cache/ # 解决下载慢的问题:将包放在cache路径(~/.pyenv/cache/)下

使用pyenv

local命令

local命令切换当前目录及其子目录的python版本, 可以通过删除.python-verson恢复默认的python版本

global命令

global命名切换全局默认的Python版本,永远不要使用!!

virtualenv命令

$ pyenv virtualenv 3.5.2 chenfanlinux # 创建虚拟环境
$ pyenv versions # 查看python版本
 * system (set by /root/.pyenv/version)
  3.5.2
  3.5.2/envs/chenfanlinux
  chenfanlinux
$ pyenv local chenfanlinux # 切换环境

uninstall命令

$ pyenv uninstall chenfanlinux # 卸载某个版本(包括虚拟环境)
